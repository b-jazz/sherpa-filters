Moving Your Inbox Contents

For the current versions of Sherpa, we treat the Inbox as a temporary place to store incoming files. Anything that comes into the Inbox is filtered and moved to another location. If you store your working set of email in your Inbox, you'll need to move it to another location. This requirement will go away in a later version of Sherpa.

It is recommended that you create a folder in your Mailboxes folder that is called In or Inbox. After you've created that folder, select all of the items in the original folder and drag them into your new Inbox.
