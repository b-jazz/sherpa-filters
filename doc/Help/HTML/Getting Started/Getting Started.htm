Getting Started

In order to start using Sherpa, you need to take care of some things first. You'll need to do the following things:

  • Find out the name of your Exchange server
  • Move your Inbox to another location
  • Set up spam and non-spam samples (optional)
  • Edit Sherpa's Options

Once you've done these things, you'll be ready to write your first Sherpa filters. You should study the sample file and read up on the filter format section below.
