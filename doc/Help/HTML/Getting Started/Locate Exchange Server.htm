Locate Your Exchange Server

In Outlook, you'll need to find out where your Exchange server is.

  • In the "Tools" menu, select "E-mail Accounts..."
  • Select the "View or change existing e-mail accounts" radio button
  • Click the "Next" button
  • Select the Exchange server from the table
  • Click "Change..."
  • Write down, or remember, the server name in the "Microsoft Exchange Server" text box
  • Click "Cancel"
