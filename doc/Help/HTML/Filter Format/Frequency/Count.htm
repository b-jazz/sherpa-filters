Count

Specify how many message to ignore before accepting another message matching the give criteria.

The Frequency tag is optional when writing filters, but when it exists, it must come after the Criteria tag and before the Actions tag.

Frequency can help you avoid some high traffic email messages that might come into your mailbox. For example, maybe you have a script that runs every five minutes and emails you if something fails. Well, you wake up the next morning and have about a hundred email messages waiting for you saying the exact same thing when your script failed five minutes after you left work the night before.

Count Example

You might only want to receive every tenth message, so you'll write a filter like the one below

<Filter Name="Count Example">
  <Criteria>
    <HeaderMatch>
      <Header>Subject</Header>
      <Value>^Script Failure: </Value>
    </HeaderMatch>
  </Criteria>
  <Frequency>
    <Count>10</Count>
  </Frequency>
  <Actions>
    ...
  </Actions>
  <Stats>
    ...
  </Stats>
</Filter>
