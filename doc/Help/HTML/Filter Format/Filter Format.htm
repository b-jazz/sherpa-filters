Filter Format

The filter tag contains one entire filter, including what attributes to look for in an email message, how often to act on a matched filter, what to do when you match a message, and what statistics have been gathered about how the filter is performing.

It is very important that all of the tags within a Filter tag appear in the exact order listed above. The only exception is the Frequency tag which is optional.

See the examples section of the documentation to see full examples of filters. This is just an abbreviated version of a filter.

  </Filter>
