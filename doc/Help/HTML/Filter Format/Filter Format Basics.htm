Filter Format Basics

The Sherpa filters are expressed in an XML file. The structure is pretty simple. There is some header information, then the single <Filters> tag that encloses multiple <Filter> tags.

The filters are examined in order, top to bottom, and the first filter that matches the email message performs its actions and then no more filters are checked.

Here is the most basic Sherpa filter that you can have.

<?xml version="1.0" encoding="UTF-8"?>
<Filters Version="1" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" xsi:noNamespaceSchemaLocation="file:///C:/Program Files/IH8SPAM/Sherpa Filters/SherpaFilterSchema.xsd">
</Filters>

The filter format version that this version of Sherpa understands is anything up to and including version 1. Future versions of Sherpa might understand later versions of the filter and will be able to auto-upgrade older versions to the current version.

If for some reason, you install Sherpa outside of the standard location, or on a different drive letter, make sure you change the URL listed in the second line.
