ListMove

Use in conjunction with the MailingListMatch criteria to move a mailing list email message into a folder that is determined by the address of the mailing list.

If you are on a list named sports@yahoogroups.com and you have a filter set up, your mail will be filter into Mailboxes/YahooGroups/sports@yahoogroups.com.

ListMove Example

This action will move the message described above.

<Actions>
  <ListMove>
    <MailingListFolder>Mailboxes/YahooGroups</MailingListFolder>
  </ListMove>
</Actions>
