Move

Move a message to a new folder.

The folder specified by Destination should be relative path. It is okay if the path contains spaces. You should use a forward slash to separate folder names just like in the web world. If the path you specify doesn't exist, it will be created for you.

Move Example

Move a matched message into your In box.

<Actions>
  <Move>
    <Destination>Mailboxes/In</Destination>
  </Move>
</Actions>
