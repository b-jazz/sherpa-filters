HeaderExists

Check to see whether or not the email message contains the specified header line.

The Header tag should contain the name of the header that you are testing the existance of. Remember not to include the colon in the name of the header.

HeaderExists Example

This example will match against an email that doesn't contain a To: header. Most legitimate mail servers will generate a To: header line but leave it blank when there is nobody on the To: line. But some spam programs out there completely remove the To: header. I've seen enough spam like this to write a rule for it.

The example also uses the <Not> criteria so that we'll match when the header doesn't exist.

<Criteria>
  <Not>
    <HeaderExists>
      <Header>To</Header>
    </HeaderExists>
  </Not>
</Criteria>
