HeaderMatch

Check to see if a specific header line has a certain string or matches a given regular expression.

The Header tag can specify any possible header. Do not include the colon as part of the header name. You can have as many Value tags as you'd like. If any one of them matches the given header, the criteria will match. Regular expressions are more than welcome as part of the Value tags.

HeaderMatch Example

The following example shows you how you might catch the mail that comes from some well known companies that don't spam you and send you mail about orders you've placed.

<Criteria>
  <HeaderMatch>
    <Header>From</Header>
    <Value>@amazon.com</Value>
    <Value>@ebay.com</Value>
    <Value>@gap.com</Value>
  </HeaderMatch>
</Criteria>

This example shows you how to look for a specific subject line and uses regular expressions. Any message that has a subject line starting with "Server Reboot" will match and be filed away somewhere else. The nice thing about this match is that if anyone replaces to the message, there will be a "RE: " at the start of the subject line which won't match and the message might end up in your In box. 

<Criteria>
  <HeaderMatch>
    <Header>Subject</Header>
    <Value>^Server Reboot: .*$</Value>
  </HeaderMatch>
</Criteria>
