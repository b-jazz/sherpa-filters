BodyMatch

Matches text in the body of the email message. Even more useful when used with regular expressions.

You can have as many Value tags as you'd like. The criteria will Or the values together so that if any of values show up in an email, the criteria will consider the message a match.

BodyMatch Example

After you've filtered away all of your business email and mail from friends, you might want to use the example below to filter out some words that are very common to spam.

<Criteria>
  <BodyMatch>
    <Value>Unsubscribe</Value>
    <Value>Special Offer</Value>
  </BodyMatch>
</Criteria>

Be careful though, there is a chance that you could throw away some legitimate email with the rule above. You might want to consider using the Bayes filter instead.
