And

Combines two or more criteria. If all the sub-criteria match, then the overall criteria triggers a match.

And Example

In the example below, Joe's weekly status report will trigger a match since the mail is from Joe and the subject line starts with "Weekly Report".

<Criteria>
  <And>
    <HeaderMatch>
      <Header>From</Header>
      <Value>joe@mycompany.com</Value>
    </HeaderMatch>
    <HeaderMatch>
      <Header>Subject</Header>
      <Value>^Weekly Report</Value>
    </HeaderMatch>
  </And>
</Criteria>
