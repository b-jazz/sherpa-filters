Foreign Language Spam

I see a fair amount of spam coming with either Chinese or Japanese characters in it. I notice that these emails have some encoding in the Subject line that looks like it escapse some character values with an equals sign. Anyway, there are a lot more equal signs than one would normally see in any other message. This filter will look subject lines that have an '=', some more characters, another '=', more characters, etc. There are several more ways to write this regular expression and to tighten it down, but this has done the trick for me.

Example

<Filter Name="SPAM: Foreign Language Spam" Enabled="true" Tag="Spam">
  <Criteria>
    <HeaderMatch>
      <Header>From</Header>
      <Value>.*=.*=.*=.*=.*=.*=</Value>
    </HeaderMatch>
  </Criteria>
  <Actions>
    <Move>
      <Destination>Mailboxes/Spam</Destination>
    </Move>
  </Actions>
  <Stats>
    <TestCounter>0</TestCounter>
    <MatchCounter>0</MatchCounter>
    <FilterTime>0</FilterTime>
    <LastMatchTime></LastMatchTime>
  </Stats>
</Filter>
